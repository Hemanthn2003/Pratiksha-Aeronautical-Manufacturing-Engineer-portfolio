# ✈️ Pratiksha Aeronautical Manufacturing Engineer Portfolio

A modern aerospace-themed portfolio website designed for showcasing manufacturing engineering expertise, Airbus production operations, MES systems, MBOM & Routing creation, industrialization, and aerospace manufacturing workflows.

The portfolio features a cinematic storm-themed animated UI with realistic clouds, lightning effects, responsive layouts, smooth animations, and modern user experience.

